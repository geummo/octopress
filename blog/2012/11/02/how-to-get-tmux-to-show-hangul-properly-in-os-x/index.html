How to Get Tmux to Show Hangul Properly in OS X

Nov 2nd, 2012

주로 회사 업무 상 영어로 개발작업이 이뤄지다보니 느끼지 못했는데

tmux에서 한글로 표시된 파일이나 디렉터리를 보면 아래 그림처럼 초성만 보이는 것을 볼 수 있습니다.

이런.. 어쩌란 말이냐.

몇시간 동안 구글링으로 찾아보니 일본쪽에서 이미 패치를 만들었더군요.

패치가 적용된 홈브루 포뮬러를 가진 대체 소스에 “탭”합니다.

brew tap waltarix/customs

설치된 tmux를 일단 제거합니다.

brew uninstall tmux

패치가 적용된 tmux를 다시 설치합니다.

brew install waltarix/customs/tmux
