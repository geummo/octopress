(MAC) OS X의 Tmux에서 한글 제대로 보기

Nov 2nd, 2012 | Comments

요즘 OS X + iTerm2 + tmux + Vim 조합으로 개발환경을 꾸미고 있습니다. 하지만 역시 한글환경에는 조금 문제가 있네요.

일단 tmux에서 한글로 된 파일이나 디렉터리를 보면 아래 그림처럼 초성만 보입니다.

헐;;;

다행히도 패치를 적용하면 한글을 제대로 볼 수 있습니다.

tmux에서 한글 제대로 보기

패치가 적용된 홈브루 포뮬러 (Homebrew Formula)를 가지고 있는 대체소스에 “탭 (brew tap) 합니다.

brew tap waltarix/customs

설치된 tmux를 일단 제거합니다.

brew uninstall tmux

패치가 적용된 tmux를 다시 설치합니다.

brew install waltarix/customs/tmux

다시 보면 아래 그림처럼 한글이 제대로 표시되는걸 볼 수 있습니다.
